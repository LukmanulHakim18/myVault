# Universal Payment Gateway (UPG) Team

## Overview
UPG adalah backend payment platform untuk aplikasi MyBB yang menangani seluruh proses pembayaran untuk berbagai layanan Bluebird Group.

## Services
- **Payment Processing Service**: Menangani berbagai metode pembayaran
- **Transaction Service**: Manajemen transaksi dan reconciliation
- **Wallet Service**: Digital wallet management
- **Invoice Service**: Invoice generation dan tracking

## Tech Stack
- Language: Go
- Architecture: Microservices
- Communication: REST API, gRPC, Event-driven
- Integration: Multiple payment gateway providers
