# Meta Reservation Gateway (MRG) Team

## Overview
MRG adalah backend platform untuk aplikasi MyBluebird yang menangani seluruh proses reservasi dan booking untuk berbagai layanan transportasi Bluebird Group.

## Services
- **Booking Service**: Menangani proses booking kendaraan
- **Reservation Service**: Mengelola reservasi jangka panjang
- **Vehicle Service**: Manajemen fleet dan vehicle availability
- **Notification Service**: Multi-channel notification system
- **ID Generator Service**: Two-level ID generation system untuk booking & orders

## Tech Stack
- Language: Go
- Architecture: Microservices
- Communication: REST API, gRPC, Event-driven
